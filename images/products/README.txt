Product images go in this folder.

Naming convention:
  {product_id}_{filename}.jpg
  e.g. p001_battery.jpg

After uploading images via the Admin Panel:
  1. Export products.js from Admin → Settings → Export JSON
  2. Replace website/js/products.js with the exported file
  3. Upload new images to this folder via FTP
  4. Re-deploy

Image recommendations:
  - Format: JPG or PNG
  - Size: 800x600px minimum
  - Max file size: 2MB
  - Ratio: 4:3 preferred
